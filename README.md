# tailwind-admin-dashboard
This is a FREE admin dashboard made with Tailwind CSS. It includes:

Demo [here](https://yzipperer.github.io/tailwind-admin-dashboard/)

```
  - Top navigation bar
  - Side navigation bar
  - grid container
  - Total YTD Sales Overview
  - Dashboard Widgets:
    - Total YTD Orders
    - Average Order Total
    - Total Products
    - Total Inventory
```
